A specialized Relationship Management Dashboard built with React.

  Centralized Logic (Context API)
    Modern Interface & Navigation
   Dynamic Nav: Sleek navbar with real-time active state indicators.
   Responsive Design: Mobile-first layout optimized for all devices.
   Deep-Dive Profiles (Dynamic Routing)
   Dedicated profiles using URL parameters and async data fetching.
   Empty States: Graceful UI handling for accounts with no prior history.

   
Useing:

    Category              Technology

    Framework             React.js (Vite)

    Routing               React Router v7

    Styling               Tailwind CSS + DaisyUI

     State                Context API

    Charts                Recharts

    Alerts                React-Toastify

    Loaders               React-Spinners
